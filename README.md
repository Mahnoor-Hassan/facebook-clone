# Facebook Clone

A fully responsive **Facebook Clone** built using **HTML, CSS, and JavaScript**. This project recreates the modern user interface of Facebook, focusing on responsive design, clean layouts, and an interactive user experience. It was developed as a front-end practice project to strengthen web development skills and understand how social media platforms are structured.

## 📌 Project Overview

The Facebook Clone replicates the appearance and basic functionality of the Facebook homepage. The goal of this project is to practice creating professional UI components, responsive layouts, and dynamic interactions while following modern web development practices.

Although this project is not connected to Facebook's backend services, it demonstrates how a real-world social media interface can be designed using front-end technologies.

## ✨ Features

- Responsive design for desktop, tablet, and mobile devices
- Modern Facebook-inspired user interface
- Navigation bar with search functionality
- Sidebar navigation
- News feed layout
- Post creation section
- Stories section
- Friend suggestions
- Like, Comment, and Share buttons
- Profile cards
- Hover effects and smooth transitions
- Clean and organized code structure

## 🛠️ Technologies Used
